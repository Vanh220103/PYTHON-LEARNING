TEST CASE 5: Credentials Case Sensitivity
═══════════════════════════════════════════════════════════
ID:              TC_LOGIN_005
Title:           Verify login is case-sensitive
Module:          Authentication
Priority:        MEDIUM
Type:            Functional

Precondition:
- Valid credentials: standard_user / password123
- Test with different case combinations

Test Steps:
1. Enter "STANDARD_USER" (uppercase) in Username field
2. Enter "PASSWORD123" (uppercase) in Password field
3. Click "Login" button
4. Check if login succeeds or fails

Expected Result:
- Login fails
- Error message: "Username and password do not match any user in this service"
- OR system accepts (depending on design)

Actual Result:
- Login fails
- Error message: "Username and password do not match any user in this service"

Status:          [PASS / FAIL]
